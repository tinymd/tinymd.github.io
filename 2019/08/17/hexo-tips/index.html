一.准备工作#

1. 了解Hexo#

Hexo是高效的静态站点生成框架，基于Node.js。 通过 Hexo 你可以轻松地使用 Markdown 编写文章，除了 Markdown 本身的语法之外，还可以使用 Hexo 提供的标签插件来快速的插入特定形式的内容。

2.在本地安装Hexo#

直接参考Hexo官方文档

2.搭建博客#

3.优化网页样式布局#

如此图中的间距，如果想要缩小或增大,具体有以下步骤：

  1. 在浏览器页面按下F12调出调试页面（火狐和Chrome的调试页面按键都是F12）
  2. 点小箭头定位元素，调试CSS代码到自己想要的效果
  3. 将修改的CSS代码Copy到custom.styl

我的custom.styl位置在E:\PersonalBlog\myHexoBlog\themes\next\source\css_custom.styl

当然，字体等属于样式范畴内的东西都是可以这样改的。如果目前看到这些不甚理解，强烈建议看看Html和CSS的官方文档，基础知识没有办法绕开和速成，建议把基础知识快速阅读完，建立基本认知，具体记忆在结合实践会较为牢固，推荐不到1小时可以将两者的基础看一遍。

4.hexo插件#

当你需要什么神奇的功能时，可以考虑一下hexo的插件plugins。在官网上找到对应插件，点击进入github页面，readme会告诉你怎样使用这个插件，一般都很简单的。

如本文就使用了hexo-toc插件，可以在文本前插入一个目录。
