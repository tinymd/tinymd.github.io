##
实验楼学习Linux

传授是记忆的最佳方式。

  1. 安装了VMware Tools，具体过程如下: 点击菜单栏中”虚拟机”-“安装VMware Tools”，然后会发现系统中多了个vmware光驱一样的东西，点进去解压里面的压缩包，然后运行”sudo ./vmware-install.pl”。之后一路默认就好。
  2. 发现linux下用vi编辑是什么反人类设计呀！！！！！ i-进去是插入模式，你如果要回车一定不能再鼠标还在单词上回车，一定得在空白处，然后插入模式下是无法删除的，如果要删除推到命令模式下用x命令删除单个字符。
  3. sudo apt-get update出了无法访问localhost:80的问题，不懂什么原因，但网上找了个CSDN的博客解决了这一问题。 博客原网址：https://blog.csdn.net/lm409/article/details/53939990/
    1. 先关闭所有源
    2. rm -rf /var/lib/apt/lists/*删除/var/lib/apt/lists 所有文件
    3. 修改/etc/apt/sources.list文件。将源改成163的或者阿里的。(注意版本匹配：16.1->yakkety;16.04->xenial;15.10->willy;14.04->trusty;12.04->precise) deb http://mirrors.163.com/ubuntu/ xenial main restricted universe multiverse deb http://mirrors.163.com/ubuntu/ xenial-security main restricted universe multiverse deb http://mirrors.163.com/ubuntu/ xenial-updates main restricted universe multiverse deb http://mirrors.163.com/ubuntu/ xenial-proposed main restricted universe multiverse deb http://mirrors.163.com/ubuntu/ xenial-backports main restricted universe multiverse deb-src http://mirrors.163.com/ubuntu/ xenial main restricted universe multiverse deb-src http://mirrors.163.com/ubuntu/ xenial-security main restricted universe multiverse deb-src http://mirrors.163.com/ubuntu/ xenial-updates main restricted universe multiverse deb-src http://mirrors.163.com/ubuntu/ xenial-proposed main restricted universe multiverse deb-src http://mirrors.163.com/ubuntu/ xenial-backports main restricted universe multiverse
    4. 再次sudo apt-get update成功

以下是通过学习实验楼里Linux基础入门（新版）所得知识： 学习网址：https://www.shiyanlou.com/courses/1

pwd –显示当前目录 “TAB” -自动补全命令 通配符-‘*’.’?’.’{c1..c2}’ touch – 新建文件 rm – 删除文件

man – 获取命令的说明和使用方式 /<要搜索的keyword>，n-下一个，shift+n-上一个，k,j-上下一行, space -翻页 单纯参数查询用 --help 需要更详细的可以用info，这个命令里面具体有什么没怎么看。

给留的第4次作业是下载一个banner的软件包，作用是输出图形字符。 两行命令： Sudo apt-get update Sudo apt-get install sysvbanner Linux自带一个printerbannner,还有另外两种字体也下载:toilet，figlet
